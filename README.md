# PMD88v33PV
# PMD88 Custom Hook Program

## 概要
このプロジェクトは、PMD88 V3.3専用のカスタムフックプログラムです。PMD88単体に対して特定の機能を追加しています。このプログラムは、音楽再生やリズム音源の制御を行うためのアセンブリ言語で書かれています。

## 特徴
- PMD88の元のコードをフックして、自プログラムに機能を追加
- 音名の表示やリズム音源の制御
- チャンネルのミュート・再生機能
- タイマーによる音色の非表示処理

## 使用方法
1. ソースコードをアセンブルして、PMD88にロードします。
2. プログラムを実行すると、音楽再生やリズム音源の制御が行われます。
3. キー入力により、各チャンネルのミュートと再生を切り替えることができます。
